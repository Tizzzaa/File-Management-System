# File-Management-System
A basic file management System for C, able to do many different actions on files.


You are able to create and delete files in the same folder that your application is running in.

### Set up

I am running and compiling my C code on Visual Studio Code and using the Win32 C configuration.

### Functions

##### File Operations

1. Create File

A file is created in your current directory with the name you specify

2. Copy File

A file of your choosing is identically copied into the same directory

3. Delete File

If a file of your choice exists in your directory, it gets deleted

4. Show File

If a file of your choice exists in your directory, it gets outputed in its entirety

##### Line Operations

1. Append File Line

If a file of your choice exists in your directory, it will appened a string of your choice to the end of your file

2. Delete line

If a file of your choice exists in your directory and the line of your choice is in that file, that line gets deleted

3. Insert file line

If a file of your choice exists in your directory, it will insert a string of your choice into a line of your choice

4. Show Line

If a file of your choice exists in your directory and the line of your choice is in that file, that line gets ouputted

##### General Operations

1. Show Change Log

A seperate file that keeps track of all the updates you have completed

2. Show Number of Lines

If a file of your choice exists in your directory, it will display the number of lines in that file

##### Special added Operations

1. Copy line from file to another

If a file of your choice exists in your directory and the line of your choice is in that file, it will copy that line to another file of your choice give that file exists

2. Display siz of a File

If a file of your choice exists in your directory, it will display the size of said file.
